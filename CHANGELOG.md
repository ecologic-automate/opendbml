# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]
### Added
- Nuxt 3 application scaffold in `app/`.
- Local-first storage via IndexedDB (`localforage`).
- Diagram/table drag improvements and relationship line anchoring.
- Drag-to-connect relationship creation with DBML updates.
- Undo/redo support in editor state.
- OSS governance files (`CONTRIBUTING`, `SECURITY`, templates).

### Changed
- Project renamed to EcoSchema.
- Build/run focus moved to `app/`.

### Removed
- Legacy Quasar/Vue app path (`web/`) and unused API placeholder (`api/`).
