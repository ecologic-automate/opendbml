# Open DBML

Open DBML is an open-source database diagram editor with a local-first workflow.

## Current App
The active application lives in `app/` (Nuxt 3 + Vue 3).

## Run Locally
```bash
cd app
npm install
npm run dev
```

The dev server binds to `0.0.0.0:3000`.

## Build
```bash
cd app
npm run build
npm run preview
```
